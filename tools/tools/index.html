    VSCode + markdown-preview-enhanced¶

    文書作成にVSCode + markdown-preview-enhancedを利用する。そのメモ。

    準備¶

    VSCodeに機能拡張をインストールする。markdown-preview-enhanced Paste Image PlantUMLをインストール。 Graphvizをインストール。

    設定¶

    markdown-preview-enhancedの"markdown-preview-enhanced.enableScriptExecution"をtrueにする。セキュリティの関係上、ワークスペース単位がいいかも。 Paste Imageの画像の保存がデフォルトではカレントなので、/imgに変更する。

    見た目の変更はMarkdown Enhanced Preview : Cutomize CSSで行う。

    チートシート¶

    数式¶

    数式を書く。Katexが使えるので$...$や$$...$$で描ける。 例：$$x = \frac{-b \pm \sqrt{b^2-4ac}}{2}a$$

    x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}

    外部ファイルを参照¶

    @import "file.name"で外部ファイルを参照できる。

    • 画像ファイルはイメージとして取り扱われる
    • .csvはテーブルに変換される
    • .mermaidはmermaidによってレンダリング
    • .plantuml, .pumlはPlantUMLによってレンダリング
    • .htmlは直接埋め込み
    • .jsは<script src="javascript.file"></script>として埋め込み
    • .cssは<link rel = "stylesheet" href = "css.file">として埋め込み
    • .mdふぃあるは直接埋め込み
    • 他のファイルはコードブロックとしてレンダリング
