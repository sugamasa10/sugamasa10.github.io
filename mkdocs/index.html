    MkDocs¶

    ドキュメントサーバー¶

    個人的な非公式ドキュメント類を作成＆整理＆公開する。HTMLでゴリゴリ書いてもいいけど、どう考えても面倒なので他の方法を検討する。

    HTMLの変わりではないが、簡単に書くための言語として軽量マークアップ言語と呼ばれるものがある。この軽量マークアップ言語からHTMLに変換することで、整理と公開を簡単にする。軽量マークアップ言語ではおそらく次のようなものが有名。

    • markdown
    • asciidoc
    • sphinx

    今回は一番簡単なmarkdownを選択した。ただしmarkdownは方言が多すぎるという問題があるので注意する。それでも色々なところで使われているので学習コストは一番低いだろう。

    markdownで記載されたファイルをhtmlに変換する。単に変換するだけなら色々あるけど、せっかくなのでカッコよくWebページとして公開したい。ここにも種類が色々あるが、導入が簡単そうなmkdocsを選択。

    インストール¶

    mkdocsはpythonなのでpipでインストールする。ついでにテンプレートや機能拡張もインストールする。

    pip install mkdocs
pip install mkdocs-material
pip install python-markdown-math
pip install mdx_truly_sane_lists
pip install pymdown-extensions
pip install mkdocs-wavedrom-plugin
pip install plantuml_markdown
pip install fontawesome_markdown

    準備が整ったのでmkdocs new projectnameでプロジェクトを作成。docsディレクトリにmarkdownファイルを入れて、mkdocs buildでビルドすとsiteディレクトリHTMLができる。このファイルを公開する。なおビルドせずに内容を確認するにはmkdocs serveとする。
